Bienvenue au site officiel de Visual Learning. Visual learning est une plateforme online créée pour aider les étudiants à comprendre les différents modules

Structures des fichiers et structures des donnees. SFSD est un module etudié en 2cp à ESI qui comporte sur les différents types des fichiers, leurs organisations et leurs fonctionnements.

Fichiers Simples

Le chapitre des fichiers simples traite les representations des fichiers comme un tableau ( T ) ou une liste ( L ) , ordonné ( O ) ou non ordonné ( Ob ), avec des enregistrements de taille fixe ( F ) ou variable ( V ) avec ou sans chevauchement ( C ou Cb).
